{{siteName}} - Site Status

Accepting Donations from the Public:
A site not accepting donations is still listed.
Distributing Donations to the Public:
A site is distributing donations if supplies are being provided to the local community and general public.
Site Type:
Distribution sites supply items primarily to the community.
Supply hubs are large warehouses.
Publicly Visible:
Impacts search result display only.
A publicly visible site is shown in all search results to everyone.
A not publicly visible site is only shown to users that are logged in.
The backend dispatch systems and data sharing with partners will work the same whether a site is publicly listed or not.
Site is Active:
An inactive site is no longer listed and is effectively removed from WNC-supply-sites.com
